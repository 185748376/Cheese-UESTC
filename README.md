# “芝士” Cheese-UESTC

> 本小程序是用于连接我电的**校内新闻网**和**图书馆**，方便同学们随时随地了解校园动态，掌握最新通知，查阅图书借阅详情。APP界面设计风格简约，包含前端展示UI和后端服务器部分，此仓库为客户端源码。
